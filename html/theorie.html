  • À savoir:

    Les suites décrivent des phénomènes qui varient par rapport à une variable n prenant des valeurs discrètes, c'est-à-dire que la variable appartient aux entiers naturels (0, 1, 2, 3...). Au contraire, les fonctions décrivent des phénomènes qui dépendent d'une variable x définie dans un intervalle continu, c'est-à-dire que x appartient aux réels (0.123, √2, ...).

    Connaissances nécessaires:

    • Définition d'une suite (un) en fonction de n ou par récurrence
    • Sens de variations
    • Suites aritmétiques ( un+1 = un + r )
    • Suites géométriques ( un+1 = un ∙ q )
    • Exercices avec des suites arithmético-géométrique et auxiliaire
    • Uniquement S: Somme de suites ( ∑ui = u0 + u1 + u2 + ... + un = ? )
